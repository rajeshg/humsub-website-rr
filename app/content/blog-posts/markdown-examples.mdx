# Markdown Examples

This post demonstrates various markdown features that you can use in your blog posts.

## Headings

You can use different levels of headings:

# H1 Heading

## H2 Heading

### H3 Heading

#### H4 Heading

## Text Formatting

You can use **bold**, _italic_, and `code` formatting.

## Lists

### Unordered Lists

- Item 1
- Item 2
  - Nested item 1
  - Nested item 2
- Item 3

### Ordered Lists

1. First item
2. Second item
3. Third item

## Code Blocks

You can write inline code like `const hello = "world"` or use code blocks:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Blockquotes

> This is a blockquote. You can use it to highlight important information or quotes.
